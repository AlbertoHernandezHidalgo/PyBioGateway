# BioGatewayQuery

A Python package to perform SPARQL queries and exploit the data available on BioGateway.

## Table of Contents

- [Installation](#installation)
- [Contributing](#contributing)

## Installation

To install the package, use pip:

```bash
pip install BioGatewayQuery
```
